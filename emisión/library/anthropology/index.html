THESIS

포옹의 인류학

인간은 왜 안는가. 아기는 안겨야 살고, 연인은 안아야 사랑하고,
죽은 자는 안겨서 떠난다. Abrazo는 인간의 원시 프로토콜.

탱고의 포옹 = 완전한 surrender
리드 + 팔로우 = 상호 의존의 계약

침묵의 인류학

탱고는 말하지 않는다. 음악만 있고, 몸만 있다.
이것은 언어 이전의 소통. 본능의 영역.

채팅 → 데이트 → 키스 → ...
탱고: 포옹 → 침묵 → 몸 → 끝

역할의 인류학

리드와 팔로우. 이것은 젠더가 아니다.
제안과 수용. 질문과 응답. 권력이 아닌 대화.

Leader = 방향을 제안하는 사람
Follower = 제안을 해석하는 사람
둘 다 능동적. 둘 다 창조적.

KEY CONCEPTS

Conexión 두 몸이 하나의 축을 공유
Intención 움직이기 전의 에너지 전달
Escucha 상대의 몸을 듣는 것
Entrega 자기를 내어주는 것
Presencia 지금 여기에 완전히 존재
