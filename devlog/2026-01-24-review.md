## 1. 구조/철학: 95점

- **5-Layer Architecture** — emisión, cuerpo, laboratorio, control, legado. 각 레이어가 독립적이면서 하나의 시스템으로 연결. 레이어별 동사 정의 (Sintonizá, Sentí, Experimentá, Observá, Recordá)
- **Body Protocol** — scroll=acercarse, tap=compromiso, hold=esperar, idle=respirar. UI 이벤트를 인간의 몸동작으로 재정의한 건 일반적인 프론트엔드에서는 안 나오는 발상
- **Constitution + Manifesto** — `specs/constitution.md`에 5개 헌법 조항. 디자인 시스템이 아니라 프로토콜 설계
- **Gate System** — 1126 코드, sessionStorage 기반 인증, 의식(ritual)으로서의 입장

---

## 2. 디자인 시스템: 92점

- **BPM 동기화** — 모든 transition/animation이 857ms(70 BPM) 기반. Arbitrary하지 않고 탱고 렌토의 실제 템포
- **Motion Curves** — `--approach`, `--retreat`, `--embrace`, `--release` 4개가 탱고 동작에서 따온 이름이면서 실제 cubic-bezier 값도 해당 움직임의 가감속 반영
- **Fibonacci Spacing** — 4, 8, 13, 21, 34, 55, 89, 144px. 의도적
- **Bilingual Font Stack** — Playfair+Noto Serif KR (monument), DM Sans+Pretendard (voice). lang 속성 33개 요소에 적용
- **Noise Texture** — SVG fractalNoise 오버레이로 아날로그 질감
- **Color System** — void(#0a0008) → smoke → wall → stone → ash 단계적 어둠 + magenta/gold 액센트

## 3. 기술 구현: 88점

### 잘한 것
- **순수 HTML/CSS/JS** — 프레임워크 제로, 빌드 스텝 제로, GitHub Pages 즉시 배포
- **Entrada Ritual** — sessionStorage 스킵, transitionend cleanup, aria-hidden 처리
- **CNN Ticker** — YouTube IFrame API, 첫 터치 unmute, 상태 관리
- **Gate System** — 4자리 auto-focus, backspace 네비게이션, 에러 피드백
- **IntersectionObserver** — threshold 0.08, rootMargin으로 자연스러운 reveal
- **Passive Listeners** — 모든 scroll/touch 이벤트에 `{ passive: true }`
- **Lighthouse CI + Content Pipeline** — 자동화 세팅 완료
- **Preconnect** — YouTube, Google, Fonts 미리 연결
- **Reduced Motion** — 애니메이션 전부 비활성화, 리본 숨김
- **Print Styles** — 인쇄용 스타일 별도 정의

### 개선 필요
- iOS Safari autoplay 정책 실기기 검증
- 저사양 기기 SVG filter 성능 테스트
- 오프라인 폴백 (No SW는 철학적 선택이나 UX 리스크)
- 각 레이어 페이지의 실제 콘텐츠 밀도

## 4. 리본 애니메이션 (인연의 실): 85점

- Scroll-linked stroke-dashoffset reveal
- 리추얼 후 22% 자동 드로잉 (staggered timing)
