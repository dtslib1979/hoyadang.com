움직임이 프로토콜이 되는 곳. 제스처를 설계한다.

GESTURE ENGINE v1.0

GESTURE 001

Ocho

기본 움직임. 발로 그리는 무한 도형. 8자.

앞으로, 뒤로. 피벗과 함께.
{ type: "figure", beats: 4, pivot: true }

GESTURE 002

Caminata

의식적인 걸음. 도착이 아닌 존재를 위한 걸음.

가슴으로 연결. 무게 전체 이동.
{ type: "locomotion", beats: 2, connection: "chest" }

GESTURE 003

Parada

힘이 아닌 존재감으로 멈추게 하기. 발이 "여기"라고 말한다.

리드 필요. 1박.
{ type: "interruption", beats: 1, requires: "lead" }

GESTURE 004

Sacada

상대가 남긴 공간을 채우기. 밀지 않고.

에너지 전이. 다리 접촉.
{ type: "displacement", beats: 2, energy: "transfer" }

GESTURE 005

Volcada

함께 쓰러지되 넘어지지 않기. 신뢰의 움직임.

공유 축. 신뢰 필수.
{ type: "off-axis", beats: 3, trust: "required" }

GESTURE 006

Gancho

걸기. 빠르고, 정확하고, 친밀하게.

폭발적 에너지. 다리 감기.
{ type: "hook", beats: 1, energy: "explosive" }

GESTURE 007

Boleo

놓아버린 다리. 공중에서 해방되는 에너지.

원형 모멘텀. 엉덩이에서 시작.
{ type: "whip", beats: 2, momentum: "circular" }

GESTURE 008

Barrida

상대의 발을 쓸어주기. 밀지 않고 안내하며.

발 대 발 접촉. 부드러운 힘.
{ type: "sweep", beats: 2, force: "gentle" }
