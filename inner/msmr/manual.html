💻 Local 모드 (카메라 원본용)

1GB 넘는 카메라 원본 파일을 자를 때 사용해요.

왜 Local 모드?
• 파일 크기 제한 없음 (1GB+ OK)
• 인터넷 업로드 불필요
• GitHub 토큰 필요 없음

이럴 때 진짜 유용해요

출사 / 여행 / 일상 촬영

1. 밖에서 촬영
2. 촬영 직후 "여기 좋았다" 기억날 때
→ 핸드폰으로 컷 포인트 입력
3. 명령어 미리 생성해둠
4. 집 와서 붙여넣기만

💡 핵심: 현장에서 기억날 때 메모하면, 나중에 "어디가 좋았더라?" 고민 안 해도 됨

배속 기능 (v3.1)

배속 설명 처리 시간
1.0x 원본 속도 (권장) 빠름 (무손실)
0.5x 슬로우 모션 느림 (리인코딩)
0.75x 약간 느리게 느림
1.25x~2.0x 빠르게 느림
⚠️ 배속 변경 시 주의

1.0x가 아니면 리인코딩이 필요합니다.
1GB 파일 기준 5~15분 소요.
원본 속도(1.0x)는 복사만 하므로 수 초 만에 완료.

FFmpeg 설치

Local 모드를 사용하려면 PC에 FFmpeg가 설치되어 있어야 해요.

운영체제 설치 방법
Windows winget install ffmpeg 또는 공식 사이트
Mac brew install ffmpeg
Linux sudo apt install ffmpeg
Windows 사용자

PowerShell 또는 명령 프롬프트(cmd)에서 실행하세요.
영상 파일이 있는 폴더에서 터미널을 열어야 합니다.

✈️ Remote 모드 (v3.0)

핸드폰에서 명령어 → PC에서 자동 실행!

이럴 때 유용해요

출사/여행 중 → 컷 포인트 입력 → Telegram 전송
↓
집 PC가 미리 처리 (1GB = 수 분)
↓
집 도착하면 이미 완료!
어떻게 작동하나?
1. PC에서 Telegram 봇 실행 (24시간)
2. 핸드폰에서 "Telegram 전송" 클릭
3. PC가 자동으로 ffmpeg 실행
4. 완료되면 알림 수신

❓ 자주 묻는 질문

Q. 다운로드한 영상 화질이 안 좋아요
A. 원본 유튜브 영상 화질 그대로입니다. MSMR은 화질을 낮추지 않습니다.
Q. 처리가 너무 오래 걸려요
A. 보통 2-5분입니다. 10분 넘으면 새로고침 후 다시 시도하세요.
Q. "토큰이 만료되었습니다" 오류
A. GitHub에서 새 토큰을 발급받아 다시 입력하세요.
Q. 자른 영상은 어디에 저장되나요?
A. GitHub Actions 서버에 24시간 보관됩니다. 링크 클릭해서 다운로드하세요.
Q. 쇼츠용 세로 영상으로 만들고 싶어요
A. 현재는 원본 비율 유지만 지원합니다. 세로 크롭은 CapCut 같은 앱에서 하세요.
Q. 저작권 문제 없나요?
A. 본인 영상만 사용하세요. 남의 영상 무단 사용은 저작권 위반입니다.
